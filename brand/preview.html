How to read this

  • Section 4 (monochrome) is the binding gate. The Finder toolbar template variant strips color — if a variant's cursor disappears here, the full-color version isn't useful in toolbar context.
  • Section 5 (favicon at 14px) is the worst-case web context. Anything that becomes a featureless rectangle here will look broken on Twitter / browser tabs.
  • Variants: baseline = thin I-beam line · ibeam = classic with serifs · block = terminal block · bold = thicker line · lowleft = repositioned
  • Pick a winner (or two for further iteration). If none survive section 4 cleanly, we go to fallback concept 10.
